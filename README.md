# 국민대학교 시각디자인학과 위키 프로토타입

서버 없이 위키 구조, 문서 탐색, 편집 경험을 먼저 테스트하기 위한 정적 프로토타입입니다.

## 파일 구조

- `index.html`: 전체 화면 구조
- `styles.css`: 모바일/데스크톱 반응형 스타일
- `app.js`: 가짜 API, 문서 렌더링, 편집 기능
- `data/wikiData.js`: 실제 문서 원본 데이터

## 사용 방법

1. `index.html`을 브라우저에서 열면 바로 동작합니다.
2. 문서 원본을 직접 고치려면 `data/wikiData.js`의 `pages` 배열을 수정하세요.
3. 화면에서 편집 후 저장하면 브라우저 `localStorage`에만 저장됩니다.
4. 원본으로 되돌리고 싶으면 화면의 `로컬 편집 초기화` 버튼을 누르세요.

## 프로토타입에서 구현한 것

- 위키형 문서 목록과 최근 변경
- 검색
- 목차 자동 생성
- 내부 문서 링크
- 외부 링크
- 주석
- 목록, 인용문, 이미지, 구분선
- 편집 모드와 미리보기
- 스크롤해도 고정되는 매크로 버튼 바
- 모바일/데스크톱 대응 레이아웃

## 나중에 서버를 붙일 때

지금은 `app.js` 안의 `fakeWikiApi`가 `data/wikiData.js`와 `localStorage`를 사용합니다.

나중에 서버를 만들면 아래 3가지만 실제 API 호출로 바꾸면 됩니다.

- `fetchPages()`
- `savePage(page)`
- `clearLocalEdits()`
